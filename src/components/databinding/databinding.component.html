Databinding

There are 4 ways to bind data:
  • value: used to bind a variable from the component to its template (unidirectional)
  • property: used to bind properties on a html or angular element (unidirectional)
  • event: used to bind an event on a html or angular event (unidirectional)
  • ngModel: used to bind a variable from the component to a html or angular element (bidirectional)
